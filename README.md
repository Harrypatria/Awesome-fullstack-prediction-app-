## 🌟 Overview

**Attrition Predictor Pro** is a production-grade, full-stack HR analytics platform designed to transform employee retention from a reactive process into a proactive strategic advantage. Powered by **Gradient Boosting Machine Learning** and the **ITDO Framework** (Insights, Triggers, Decisions, Operations), it provides HR leaders with an agentic command center to predict risk, identify drivers, and execute retention tasks.

> "The future of HR isn't just about reporting what happened. It's about predicting what will happen and having the operational command center to change the outcome." — **Dr. Harry Patria**

## 🛠️ The ITDO Framework

Our platform systematically bridges the gap between data and action:

1.  **Insights 🔍**: Predictive modeling identifies "at-risk" employee segments using SHAP values for local interpretability.
2.  **Triggers ⚡**: Automated alert management based on custom risk thresholds (e.g., Risk > 70%).
3.  **Decisions 🧠**: AI-suggested interventions based on specific risk drivers (e.g., Equity Refresh for High-Potential Sales).
4.  **Operations 📋**: A task-tracking module where decisions turn into assigned, deadline-driven actions.

## ⚙️ Setup & Installation

### Prerequisites
-   Node.js (v18+)
-   Python (v3.10+)
-   Docker (optional)

### Local Development
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Harrypatria/attrition-predictor-pro.git
    cd attrition-predictor-pro
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    pip install -r backend/requirements.txt
    ```
3.  **Set up environment variables**:
    Create a `.env` file based on `.env.example`.
4.  **Run the application**:
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:3000`.

### Docker Containerization
1.  **Build the image**:
    ```bash
    docker build -t attrition-predictor-pro .
    ```
2.  **Run the container**:
    ```bash
    docker run -p 3000:3000 attrition-predictor-pro
    ```

## 💻 Running on VS Code Studio

1.  Open the project in **VS Code**.
2.  Install the recommended extensions: **ESLint**, **Prettier**, **Tailwind CSS IntelliSense**, and **Python**.
3.  Open the integrated terminal.
